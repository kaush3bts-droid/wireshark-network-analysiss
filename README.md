Network Traffic Analysis using Wireshark on Kali Linux
This project demonstrates network traffic analysis using Wireshark on Kali Linux.

  Tools Used
- Wireshark 4.6.4
- Kali Linux
- Terminal (nslookup, ping)
  
  What I Did
- Captured live network packets on eth0 interface
- Analyzed DNS traffic using nslookup
- Performed ICMP ping test to google.com
- Applied display filters (dns, icmp)
- Studied Protocol Hierarchy Statistics

 Protocols Analyzed
 ARP, DNS, ICMP, UDP, IPv4
